# srscli - A command line client + library for SRS FEC & VMM3

## Building

```
git clone https://github.com/bl0x/srscli
cd srscli
make
```

## Running

```
./build_*/bin/srscli
```

Note: `srscli` assumes that the FEC has an IP address of 10.0.0.2.
